# UI仕様（v1）

本章はUI-REQ-001〜003の補足仕様である。

## 表示モード
- TRAIN_ONLY: 電車（上り2/下り2）
- TRAIN_AND_BUS: 電車（上り2/下り2）+ バス（次2）
- BUS_ONLY: バス（次2）

## 表示例
### TRAIN_ONLY
```
長岡京  東海道線  更新 07:01
↑ 07:05 普通 米原行 / 07:18 新快速 近江塩津行
↓ 07:12 新快速 網干行 / 07:25 普通 姫路行
```

### TRAIN_AND_BUS
```
野洲  更新 07:01
↑ 07:05 普通 米原行 / 07:18 新快速 近江塩津行
↓ 07:12 新快速 網干行 / 07:25 普通 姫路行
Bus → 村田  07:30 / 07:50
```

### BUS_ONLY
```
村田付近  更新 07:01
Bus → 野洲駅  07:10 / 07:35
```

## Widget操作
- 手動更新:
  - 更新ボタンを提供する（UI-REQ-003）
