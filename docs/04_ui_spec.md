# UI仕様（v1）

本章はUI-REQ-001〜003の補足仕様である。

## 表示モード
- TRAIN_ONLY: 電車（上り2/下り2）
- TRAIN_AND_BUS: 電車（上り2/下り2）+ バス（次2）
- BUS_ONLY: バス（次2）

## 表示例
### TRAIN_ONLY
```
長岡京  東海道線  更新 07:01
↑ 07:12 京都 / 07:25 京都
↓ 07:05 米原 / 07:18 米原
```

### TRAIN_AND_BUS
```
野洲  更新 07:01
↑ 07:12 京都 / 07:25 京都
↓ 07:05 米原 / 07:18 米原
Bus → 村田  07:30 / 07:50
```

### BUS_ONLY
```
村田付近  更新 07:01
Bus → 野洲駅  07:10 / 07:35
```

## Widget操作
- 駅切替:
  - v1では「前へ/次へ」ボタン等のクリック操作で実装する（UI-REQ-002）
- 手動更新:
  - 更新ボタンを提供する（UI-REQ-003）
